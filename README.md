Веб-приложение для поиска и бронирования оптимального маршрута по типу Aviasales на Spring Boot

Использованные технологии
- Hibernate
- Spring Boot
- Spring Data JPA
- Spring Security
- Postgresql
- oAuth2
- WebSockets
- Google geocoding API
- JavaScript
- Freemarker
- HTML
- CSS
- BCryptPasswordEncoder
